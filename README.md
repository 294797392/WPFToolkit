# WPF 自定义控件开发

创建一个WPF自定义控件项目，该项目会自动生成一个名字是Themes的文件夹，并且该文件夹里有个Generic.xaml文件。Themes文件夹是一个特殊的文件夹，WPF在运行的时候，会在这个文件夹里去自动加载Generic.xaml里的资源
